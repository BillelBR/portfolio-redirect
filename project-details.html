Project: AUTOLOC - Car Rental Website

This project is a modern and responsive website developed for AUTOLOC, a professional car rental company. The platform showcases available vehicles with elegant design and user-friendly navigation to ensure an easy and efficient booking experience.

The website was built with a clean and intuitive user interface, ensuring fast performance and optimal user satisfaction across all devices.

Key features of the project include:

  • Vehicle listings with dynamic filtering and details pages
  • Online reservation form
  • Mobile-first responsive layout
  • Custom admin dashboard for managing listings
  • Photo gallery and detailed visuals for each car
